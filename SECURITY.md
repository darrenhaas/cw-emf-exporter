# Security Policy

## Supported Versions

Security fixes are handled for the latest released version on PyPI.

## Reporting a Vulnerability

Please report security issues to `mail+pypi@darrenhaas.com` rather than opening
a public issue with exploit details.

Include:

- Affected version.
- A clear description of the behavior.
- Minimal reproduction steps when possible.
- Any relevant environment details, excluding secrets and production data.

The maintainer will acknowledge the report, assess impact, and publish a fix or
mitigation guidance as appropriate.
